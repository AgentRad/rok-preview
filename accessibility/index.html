Accessibility statement

Built to be used by everyone.

Ring of Keys is committed to making this website accessible to the widest possible audience. We target WCAG 2.1 Level AA and treat accessibility as part of the work, not a checkbox.

Standards we follow

This site is designed against the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA, published by the W3C. We use these criteria as the floor, not the ceiling.

What we have built in

  • Keyboard navigation. Every interactive element is reachable and operable via keyboard. Focus rings are visible, not suppressed.
  • Skip link. A skip-to-main-content link is the first focusable element on every page.
  • Semantic structure. Landmark roles, heading hierarchy, and ARIA labels are applied throughout.
  • Color contrast. Text-to-background contrast meets or exceeds 4.5:1 (3:1 for large text). The cream-and-navy palette is checked across every component.
  • Touch targets. Tap targets meet a 44 by 44 pixel minimum on mobile.
  • Reduced motion. The site respects prefers-reduced-motion. Reveal animations and transitions are removed when users opt out.
  • Forms. Every input has a visible label, an autocomplete attribute, an inputmode where helpful, and inline error guidance.
  • Images. Every meaningful image has alt text. Decorative images are marked accordingly.
  • Live regions. Form confirmations and dynamic content use ARIA live regions so they reach screen readers.

Known issues

This is a design preview. The live WordPress build will be audited end-to-end against WCAG 2.1 AA before launch. Known items being tracked for the live build:

  • Member profile pages will be reviewed once member content is migrated. Headshots will use individualized alt text supplied by each Key.
  • Embedded podcast and video players will be tested against keyboard and screen reader controls.
  • The directory filter UI will be retested after pagination and live filtering are added.

How to give feedback

If you find an accessibility barrier, please tell us. We treat accessibility reports as priority work.

Email access@ringofkeys.org with the page URL, what happened, and what assistive technology you were using if applicable.

We aim to acknowledge accessibility reports within two business days and to ship a fix or a clear timeline within ten.

Last reviewed
